# mpv.net 自用配置文件

- 使用原生 mpv.net 播放器
- 集成进度条缩略图：[https://github.com/po5/thumbfast](https://github.com/po5/thumbfast)
- 集成漂亮的 UI 框架：[https://github.com/Samillion/ModernZ](https://github.com/Samillion/ModernZ)
- 快捷键优化

## 使用方法

- 先安装 mpv.net 播放器：[https://github.com/mpvnet-player/mpv.net/releases](https://github.com/mpvnet-player/mpv.net/releases)
- 下载本仓库所有文件，或者`git clone https://github.com/akFace/mpv.net.config.git`
- 如图所示，右键>配置>打开配置文件夹或者`Ctrl + f`快捷键打开配置文件夹
  ![image](https://raw.githubusercontent.com/akFace/mpv.net.config/master/preview/Snipaste_2026-03-16_20-34-06.jpg)

- 将所有文件复制道配置文件夹，重启播放器即可
